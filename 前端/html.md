## HTML

超文本标记语言

### What

HTML 是一种用来告知浏览器如何组织页面的标记语言，由一系列元素组成

#### 结构

```
<p> This is a p tag <p>
```

- 开始标签
- 内容
- 结束标签

嵌套元素：把元素放到其他元素之中
块级元素：以块的形式存在，即会隔行，如h、p标签
内联元素：再块级元素中并环绕文档内容，不会导致文本换行，如a标签
空元素：只有一个标签，只用来插入或嵌入一些东西，如img

#### 属性

属性包含元素额外的信息，不出现在实际内容中

- class 名称识别
- href 超链接
- title 为超链接声明额外的信息

布尔属性：没有值的属性，如disabled，用于禁止表单输入

#### 文档结构

```
<!DOCTYPE html> <!--声明文档类型-->
<html lang="zh-CN"> <!--包裹页面所有内容，也被称为根元素-->
  <head> <!--一个容器，包含在页面但不需要显示的内容，如关键词、字符集声明等-->
    <meta charset="utf-8" />
    <title>我的测试站点</title> <!--页面的标题-->
  </head>
  <body>  <!--访问页面时所有显示在页面上的内容-->
    <p>这是我的页面</p>
  </body>
</html>
```

#### 实体引用

一些字符是特殊字符，如 `>` `<` `'` `&` 等，是HTML语法的一部分，如果要使用，需要使用字符引用
如：< 通过 `&lt;` 进行引用

### 多媒体和嵌入

#### 图片

`<img src="xxx/xxx.jpg" title="xx" alt="xxx" width="400" height="400">`

title：支持信息，鼠标悬浮显示
src: 图片链接，使用相对链接或绝对链接
alt：备选文本，当图片链接错误时，显示备选文本
width：宽度
height：高度

#### 视频和音频

`<video src="" controls width="400" height="400" autoplay loop muted poster="xx.png">`

- src：链接
- controls：用户能够控制视频和音频的回放功能
- width：宽度
- height：高度
- autoplay：自动立即播放
- loop： 循环播放
- muted：播放默认关闭声音
- poster：预览图，一个图片URL

`<audio>`

和`<video>`基本相同

- 比视频播放器占空间小
- 不支持宽高属性
- 不支持预览图属性

#### iframe

可以将其他Web文档嵌入到当前文档中
`<iframe src="xxx" width="" height="" frameborder="0" allowfullscreen sandbox>`

- allowfullscreen：可以设置全屏模式
- frameborder：绘制边框，0为删除边框
- src：链接路径
- sandbox：提高安全性设置

#### 矢量图形和SVG

- 位图：使用像素网格定义，格式包括.bmp\.png\.jpg\.gif
- 矢量图：使用算法定义，格式为.svg

SVG：用于描述矢量图形的XML语言
可以通过SVG来绘制矢量图形

引入方式
- 通过`<img>`标签引入
- 通过`<svg>`元素
- 通过`<iframe>`元素

####  响应式图片

用户浏览器有不同分辨率的图片可用，这样可以根据不同的设备的屏幕大小载入合适分辨率的图片

响应式图片技术：通过让浏览器提供多个图像文件来解决不同分辨率及空间适配问题

img 新增两个属性
- srcset
- sizes
可以设置多个图片及像素大小

新增元素：
`<picture>`
